# CSV to JSON Converter

A robust Python tool for converting CSV files to JSON format with automatic type detection and flexible configuration options.

## Features

- 🔄 **Automatic Type Detection**: Converts strings to appropriate types (int, float, bool)
- 📊 **Statistics**: View row count, column count, and empty value counts
- 🎯 **Custom Delimiters**: Support for comma, semicolon, tab, and other delimiters
- 💾 **Multiple Output Options**: Save to file or get JSON string directly
- 🧪 **Comprehensive Testing**: Unit tests included for reliability
- 📦 **Easy to Use**: Simple command-line interface and programmatic API
